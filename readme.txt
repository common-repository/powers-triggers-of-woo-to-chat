Único plugin que envia mensagens WhatsApp automáticas ilimitadas para os seus clientes
Com esta aplicação você pode enviar mensagens automáticas por whatsapp integrando com diversos plugins Wordpress.
Integrações disponíveis:
* Woocomerce
* Contact Form 7
* Claudio Sanches – Correios for WooCommerce
* CartBounty – Save and recover abandoned carts for WooCommerce
* JetBooking
* Woo Reminder

- Reconquiste clientes enviando um cupom por whastsapp automático quando o pedido for cancelado;
- Crie mensagens personlizadas com mais de 30 variáveis disponíveis;
- Envie o código de rastreio automaticamente com o link para acompanhar;
- Diversas integrações com plugins terceiros;

== Como funciona ==

Powerful Auto Chat permite enviar mensagens aos seus clientes quando um status do pedido é alterado e outras situações. É preciso comprar chaves de recarca que valem por um mês.

== Recursos ==

- Autenticação com um clique, sem maiores configurações;
- Crie modelos de mensagens para cada status de pedido;
- Use os shortcuts para personalizar cada mensagem;

== Installation ==

1. Faça upload da pasta `powers-triggers-of-woo-to-chat` para `/wp-content/plugins/` diretorio
2. Ative o plug-in através do menu "Plugins" no WordPress.
3. Compre uma chave e insira no local indicado. 
4. Ative os gatilhos que desejar e personalize o teamplate de cada um.
5. Faça um teste criando um pedido .enjoy
